smallしか通してないものです。

フォルダ構成は以下の通りです。

- 2012〜2015 : 各年に開催された下記以外のコード
- atcoder : ABC/ARCのコード。他のAtCoder主催のコンテストは、年次フォルダ参照。
- codeforce : CodeForces通常ラウンドのコード。他のCodeForces主催のコンテストは、年次フォルダ参照。
- gcj : Google Code Jamのコード
- srm : TopCoderのSingle Round MatchおよびTopCoder Openのコード
- yuki : yukicoderのコード
- writer : 公開済み作問。generatorやvalidatorもあるものもあります。
